# Lamden Governance CLI

A simple node package using Lamden-Js that allows creating motions and voting on governance items

## Install

### Via NPM
```bash
npm install -g governance_js
```

### Via Repo
clone package

```bash
https://github.com/Lamden/governance_js.git
```

goto directory

```
cd governance_js
```

install

```bash
npm install -g
```

## Run

```bash
lamden_gov
```
